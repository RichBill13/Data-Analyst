# Project 3 — Financial Analysis

## Objective
Analyze financial performance across countries, products and segments.

## Dataset
- Source : Microsoft Financial Sample
- Size : 700 rows, 16 columns
- Period : 2013-2014

## Tools
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SSMS)
- Power BI

## Key Questions Answered
1. What is the profit trend over the years?
2. Which country is the most profitable?
3. Which product generates the most profit?
4. Which segment has the best Profit/Sales ratio?

## Key Insights
- Profit grew 249% from 2013 to 2014
- France is the most profitable country ($3.78M profit)
- Paseo is the top product ($4.79M profit)
- Channel Partners has the best margin at 73%
- Enterprise segment is losing money (-3% margin) — needs urgent review

## Pipeline
1. Python — data loading, cleaning, exploration, visualization
2. SQL — aggregations and business queries
3. Power BI — interactive dashboard
