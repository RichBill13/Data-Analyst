# Project 1 — Superstore Sales Analysis

## Objective
Analyze e-commerce sales data to identify key business insights.

## Dataset
- Source : Kaggle Superstore Sales
- Size : 9,800 rows, 18 columns
- Period : 2015-2018

## Tools
- SQL (SSMS)
- Google Sheets

## Key Questions Answered
1. What is the total, average, min and max revenue?
2. Which customer segment is most profitable?
3. Which are the top 5 cities by orders?
4. Which region generates the most revenue?
5. Are there months that perform better than others?

## Key Insights
- Total revenue : $2.26M
- Consumer segment dominates with $1.14M (50% of total revenue)
- West region leads with $710K
- November is the peak month every year (Black Friday effect)
- Standard Class is the most used shipping mode (59% of orders)
