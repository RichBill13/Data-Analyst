# Data Analyst

## Tools
- SQL (SSMS) — queries, joins, subqueries
- Python (Pandas, Matplotlib, Seaborn) — analysis and visualization
- Power BI — interactive dashboards and DAX measures
- Google Sheets — data cleaning and formulas

## Projects

### Project 1 — Superstore E-commerce Analysis
- Dataset : Superstore Sales (9,800 rows)
- Tools : SQL + Google Sheets
- Key insights : Total revenue $2.26M, Consumer West is the dominant segment

### Project 2 — Categories & Delivery Analysis
- Dataset : Superstore Sales
- Tools : Power BI + DAX
- Key insights : Technology leads with $827K revenue, Standard Class is the dominant shipping mode

### Project 3 — Financial Analysis
- Dataset : Microsoft Financial Sample (700 rows)
- Tools : Python + SQL + Power BI
- Key insights : 249% profit growth in 1 year, Channel Partners segment has 73% profit margin
