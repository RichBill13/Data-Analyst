# Data-Analyst

## Outils maîtrisés
- SQL (SSMS) — requêtes, jointures, sous-requêtes
- Python (Pandas, Matplotlib, Seaborn) — analyse et visualisation
- Power BI — dashboards interactifs et mesures DAX
- Google Sheets — nettoyage et formules

## Projets

### Projet 1 — Analyse Superstore E-commerce
- Dataset : Superstore Sales (9800 lignes)
- Outils : SQL + Google Sheets
- Insights : CA 2.26M$, Consumer West segment dominant

### Projet 2 — Analyse Catégories & Livraison
- Dataset : Superstore Sales
- Outils : Power BI + DAX
- Insights : Technology CA 827K$, Standard Class mode dominant

### Projet 3 — Analyse Financière
- Dataset : Microsoft Financial Sample (700 lignes)
- Outils : Python + SQL + Power BI
- Insights : Croissance 249% en 1 an, Channel Partners marge 73%
