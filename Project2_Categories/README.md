# Project 2 — Categories & Delivery Analysis

## Objective
Analyze product category performance and delivery modes across regions.

## Dataset
- Source : Kaggle Superstore Sales
- Size : 9,800 rows, 18 columns

## Tools
- Power BI
- DAX measures

## Key Questions Answered
1. Which product category generates the most revenue?
2. Which delivery mode is most used and most profitable?
3. Do categories perform better in certain regions?

## Key Insights
- Technology leads with $827K revenue (36% of total)
- Standard Class dominates with 59% of all orders
- West region outperforms all others across every category
- Consumer + West is the most profitable combination ($355K)

## DAX Measures Created
- CA Total
- Nb Commandes
- Panier Moyen
- CA West
- Croissance %
